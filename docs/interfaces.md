# Public interfaces between crates

This document enumerates the **public type signatures** at every crate
boundary. Everything listed here is part of the crate's stable-ish API —
changing it is a cross-crate refactor.

Generated by hand on the initial skeleton. Regenerate whenever a
`pub`-level type moves.

## Core infrastructure

### `rcc_errors`

```rust
pub enum Level { Error, Warning, Note, Help, Bug }

pub struct Label { pub span: Span, pub message: String, pub primary: bool }
pub struct Diagnostic {
    pub level: Level,
    pub code: Option<&'static str>,
    pub message: String,
    pub labels: Vec<Label>,
    pub notes: Vec<String>,
    pub help:  Vec<String>,
}

pub trait Emitter: Send { fn emit(&mut self, d: &Diagnostic); }
pub struct StderrEmitter;
pub struct CaptureEmitter;

pub struct Handler { /* .. */ }
impl Handler {
    pub fn with_emitter(emitter: Box<dyn Emitter>) -> Self;
    pub fn error_count(&self) -> u32;
    pub fn has_errors(&self) -> bool;
    pub fn emit(&mut self, d: &Diagnostic);
    pub fn struct_err(&mut self, span: Span, msg: impl Into<String>) -> DiagnosticBuilder<'_>;
    pub fn struct_warn(&mut self, span: Span, msg: impl Into<String>) -> DiagnosticBuilder<'_>;
    pub fn err(&mut self, msg: impl Into<String>) -> DiagnosticBuilder<'_>;
}

pub struct DiagnosticBuilder<'a>;
impl DiagnosticBuilder<'_> {
    pub fn code(self, code: &'static str) -> Self;
    pub fn primary(self, span: Span, message: impl Into<String>) -> Self;
    pub fn label  (self, span: Span, message: impl Into<String>) -> Self;
    pub fn note   (self, message: impl Into<String>) -> Self;
    pub fn help   (self, message: impl Into<String>) -> Self;
    pub fn emit(self);
}
```
